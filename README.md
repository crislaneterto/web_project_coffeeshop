# Triple Espresso

Projeto: Landing Page Triple Peaks - Café & Biblioteca

Este projeto é uma Landing Page completa desenvolvida para a cafeteria "Triple Peaks". O site foi construído para oferecer uma experiência imersiva que combina cultura (biblioteca) e gastronomia (café especial), apresentando seções de receitas, reservas e contatos.

Tecnologias Utilizadas

    HTML5 Semântico: Estruturação avançada dividida em seções (<header>, <main>, <section>, <footer>) para garantir que o conteúdo seja acessível e bem indexado.

    CSS3 Moderno:

        Metodologia BEM (Block, Element, Modifier): Utilizada em toda a folha de estilo para garantir um código escalável e modular, facilitando a manutenção de componentes repetitivos como botões e inputs.

        Flexbox & CSS Grid: Empregados para o layout complexo, como a seção de "Receitas" com vídeos e o formulário de reserva.

        Manipulação de Formas Geométricas: Uso de border-radius e z-index para criar os círculos decorativos que servem como identidade visual recorrente em toda a página.

        Estilização de Formulários: Customização de inputs de data, e-mail e número para manter a consistência visual do design.

Funcionalidades Principais

        Seção de Receitas: Galeria de vídeos incorporados (YouTube) com cartões informativos sobre métodos de preparo (Aeropress e Prensa Francesa).

        Sistema de Reservas: Formulário funcional com validação de campos obrigatórios, permitindo ao usuário escolher data, hora e número de convidados.

Responsividade Total

      Implementar um design fluído que adapte a ilustração isométrica e o formulário de reserva para visualização em tablets e smartphones, garantindo que o botão "Reservar uma mesa" seja facilmente clicável em telas touch.


Modo Leitura (Light/Dark Mode)

      Criar uma variação de tema "Noite" (Dark Mode) que utilize tons de azul ainda mais profundos, ideal para usuários que navegam no site durante períodos de estudo noturno.
